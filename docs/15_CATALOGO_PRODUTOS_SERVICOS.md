## 1. TIPOS DE VIDROS

### 1.1 Vidro Comum (Float)

**Descrição:** Composição básica do vidro, sem tratamentos especiais. Base para todos os outros tipos.

| Característica | Especificação                                        |
| -------------- | ---------------------------------------------------- |
| Espessuras     | 2mm, 3mm, 4mm, 5mm, 6mm, 8mm, 10mm, 12mm, 15mm, 19mm |
| Cores          | Incolor, Verde, Fumê, Bronze                         |
| Aplicações     | Janelas simples, vitrines, molduras, quadros         |
| Resistência    | 5x menor que temperado                               |

**⚠️ Limitações:** Não indicado para áreas de segurança. Ao quebrar, forma cacos pontiagudos.

---

### 1.2 Vidro Temperado

**Descrição:** Vidro submetido a tratamento térmico (têmpera) que aumenta resistência em até 5x.

| Característica | Especificação                               |
| -------------- | ------------------------------------------- |
| Espessuras     | 6mm, 8mm, 10mm, 12mm                        |
| Cores          | Incolor, Verde, Fumê, Bronze, Extra Clear   |
| Processo       | Aquecimento a 700°C + resfriamento rápido   |
| Resistência    | 5x mais resistente que vidro comum          |
| Segurança      | Quebra em pequenos fragmentos não cortantes |

**Aplicações:**

- Box para banheiro ✅
- Portas e janelas ✅
- Tampos de mesa ✅
- Prateleiras ✅
- Fachadas (com laminação) ✅

**⚠️ Importante:** Não pode ser cortado ou furado após a têmpera. Medidas devem ser definidas antes do processo.

**Norma:** NBR 14698 - Vidro Temperado

---

### 1.3 Vidro Laminado

**Descrição:** Duas ou mais lâminas de vidro unidas por película de PVB (Polivinil Butiral).

| Característica | Especificação                                            |
| -------------- | -------------------------------------------------------- |
| Espessuras     | 6mm (3+3), 8mm (4+4), 10mm (5+5), 12mm (6+6), 16mm, 20mm |
| Composição     | Vidro + PVB + Vidro                                      |
| Película PVB   | 0,38mm, 0,76mm, 1,14mm, 1,52mm                           |
| Cores          | Incolor, Verde, Fumê, Bronze, Colorido (PVB colorido)    |

**Benefícios:**

- Segurança: fragmentos ficam presos à película
- Proteção UV: bloqueia 99% dos raios UV
- Isolamento acústico: reduz até 40dB
- Mantém vão fechado se quebrar

**Aplicações Obrigatórias (NBR 7199):**

- Coberturas e pergolados ✅
- Guarda-corpo ✅
- Claraboias ✅
- Marquises ✅
- Pisos de vidro ✅
- Fachadas inclinadas ✅

---

### 1.4 Vidro Laminado Temperado

**Descrição:** Combinação das vantagens do temperado e laminado.

| Característica | Especificação                                    |
| -------------- | ------------------------------------------------ |
| Composição     | Temperado + PVB + Temperado                      |
| Espessuras     | 10mm (5+5), 12mm (6+6), 16mm (8+8), 20mm (10+10) |
| Resistência    | Máxima (temperado + laminação)                   |

**Aplicações Premium:**

- Guarda-corpo de alto tráfego
- Coberturas com fixação por ferragens
- Pisos de vidro transitáveis
- Fachadas estruturais

---

### 1.5 Vidro Jateado

**Descrição:** Vidro com superfície fosca criada por jateamento de areia ou processo químico.

| Característica | Especificação                             |
| -------------- | ----------------------------------------- |
| Base           | Comum, Temperado ou Laminado              |
| Opacidade      | Translúcido (permite luz, bloqueia visão) |
| Acabamento     | Fosco uniforme ou com desenhos            |
| Espessuras     | 4mm a 12mm                                |

**Aplicações:**

- Box de banheiro (privacidade)
- Divisórias de escritório
- Portas internas
- Painéis decorativos

**⚠️ Cuidado:** Superfície porosa, absorve gordura. Difícil limpeza. Preferir acidato.

---

## 2. BOX PARA BANHEIRO

### 2.1 Modelos por Configuração

#### 2.1.1 Box Frontal (Reto)

**Descrição:** Modelo mais comum. Uma folha fixa + uma porta de correr.

| Variação | Descrição                       |
| -------- | ------------------------------- |
| Simples  | 1 fixa + 1 correr               |
| Duplo    | 2 fixas + 2 correr (vão grande) |

**Medidas típicas:**

- Largura: 0,80m a 2,00m
- Altura: 1,85m a 2,10m

---

#### 2.1.2 Box de Canto (Em L)

**Descrição:** Aproveita canto do banheiro. Duas laterais com vidro.

| Variação           | Descrição             |
| ------------------ | --------------------- |
| 2 fixas + 2 correr | Portas nos dois lados |
| 2 fixas + 1 correr | Porta em um lado só   |

**Medidas típicas:**

- Lado 1: 0,70m a 1,50m
- Lado 2: 0,70m a 1,50m
- Ângulo: 90°

---

#### 2.1.3 Box de Abrir

**Descrição:** Porta abre para fora como porta convencional.

**Vantagens:**

- Abertura total do vão
- Visual clean
- Dobradiças decorativas

**Desvantagens:**

- Ocupa espaço ao abrir
- Requer banheiro maior

---

#### 2.1.4 Box Articulado (Sanfonado)

**Descrição:** Porta dobra sobre si mesma. Ideal para espaços restritos.

| Configuração | Folhas          |
| ------------ | --------------- |
| 2 folhas     | Básico          |
| 3 folhas     | Abertura maior  |
| 4 folhas     | Máxima abertura |

---

#### 2.1.5 Box Pivotante

**Descrição:** Porta gira em eixo central ou deslocado.

**Tipos:**

- Pivô central: gira 180°
- Pivô deslocado: ocupa menos espaço

---

#### 2.1.6 Box Walk-In

**Descrição:** Sem porta. Apenas painéis fixos. Entrada livre.

**Requisitos:**

- Mínimo 1,20m de largura
- Ralo linear recomendado
- Impermeabilização adequada
