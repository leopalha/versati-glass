## 1. RESUMO EXECUTIVO

### Tipos de Precificação

| Tipo         | Descrição                     | Exemplo          |
| ------------ | ----------------------------- | ---------------- |
| `PER_M2`     | Preço por metro quadrado      | Vidros, Espelhos |
| `FIXED`      | Preço fixo por unidade        | Ferragens, Kits  |
| `QUOTE_ONLY` | Sob consulta (requer medição) | Box, Portas      |

### Fórmula Geral de Cálculo

```
PREÇO FINAL = (Material + Acabamento + Instalação + Ferragens) × Multiplicadores
```
